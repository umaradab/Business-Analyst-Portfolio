# Business Analyst Portfolio - E-Commerce Funnel & Revenue Analysis

## 🎯 Project Overview
This project analyzes an e-commerce sales funnel to identify customer drop-off points, revenue drivers, and operational inefficiencies. The goal was to understand user behavior across the purchase journey and provide data-backed recommendations to improve conversion rates.

## 📊 Key Business Questions Answered
1. Where are customers dropping off in the sales funnel?
2. Which customer segments generate the highest revenue?
3. What is the monthly revenue trend?

## 🔧 Tools Used
- SQL: Data extraction, transformation, and analysis
- Excel: Data cleaning and quick visualizations
- GitHub: Portfolio documentation

## 📈 Key Insights
1. The largest drop-off occurs at the "Add to Cart" stage
2. Top 20% of customers contribute to 80% of total revenue
3. Customer retention drops significantly after the first month

## 💡 Recommendations
1. Streamline the checkout process
2. Implement a loyalty program
3. Target high-value segments with personalized offers
